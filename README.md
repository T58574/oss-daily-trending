# 🚀 OSS Daily Trending

ежедневный спам трендами гитхаба в твой телеграм канал. работает через GitHub Actions, так что сервак не нужен.

## 🛠️ как настроить

1. **создай бота:** пиши [@BotFather](https://t.me/BotFather), делай `/newbot` и забирай `BOT_TOKEN`.
2. **создай канал:** добавь туда бота как админа.
3. **узнай CHAT_ID:** 
   - если канал публичный, то это его юзернейм (например, `@my_channel`).
   - если приватный, перешли любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot) или используй `getUpdates` через браузер.
4. **настрой репозиторий:**
   - залей этот код в свой приватный (или публичный) репозиторий на гитхабе.
   - иди в `Settings` -> `Secrets and variables` -> `Actions`.
   - нажми `New repository secret`.
   - добавь `BOT_TOKEN` и `CHAT_ID`.

## 🤖 запуск
- скрипт запускается сам каждый день в 08:00 UTC.
- если хочешь проверить прямо сейчас: иди во вкладку `Actions` -> `Daily Trending` -> `Run workflow`.

👾 (lifehacks and tricks)
- если трендов слишком много, в `send_trending.py` можно поменять `repos[:10]` на любое число.
- чтобы изменить время рассылки, поправь `cron` в `.github/workflows/trending.yml`.
